# Contributing to Settlement Radio

Thanks for your interest — it means a lot.

Settlement Radio is **built in public, but not yet taking external code contributions.** The whole
station is written and built by Claude (and Claude Code), and keeping the build a clean, end-to-end
AI-authored artifact is part of the point. So for now there's no PR process for new code.

**What's welcome:**

- **Issues and ideas.** Bug reports, questions, and suggestions are genuinely useful — open an
  [issue](https://github.com/SettlementRadio/station/issues).
- **Following along.** The build happens in the open. Watch the repo, read the build log in
  [`docs/DEVLOG.md`](docs/DEVLOG.md), and follow the story on [X](https://settlementradio.com).

If that changes and we open up code contributions, this file will say so first.
